# ShadeX: Full-Stack Crypto Application 🚀

Welcome to the **ShadeX** project, a modern and high-performance cryptocurrency tracking and trading ecosystem. This repository contains both the mobile frontend and the supporting backend server.

## Project Structure

The project is divided into two main components:

- **[ShadesX.](./ShadesX.)**: The mobile application built with React Native and Expo.
- **[ShadesX-Server](./ShadesX-Server)**: The backend API built with Node.js, Express, and MongoDB.

---

## Core Features
- **Real-time Market Data**: Track live cryptocurrency prices and market trends.
- **Wallet Management**: Securely manage your digital assets and view balances.
- **Interactive Charts**: Visualize price movements with high-performance charts.
- **Safe Authentication**: Robust user registration and login system.
- **History Tracking**: Keep a detailed log of all your transactions and activities.

---

## Tech Stack Overview

### Frontend
- **Framework**: React Native with Expo
- **Styling**: NativeWind (Tailwind CSS)
- **Navigation**: Expo Router (File-based routing)

### Backend
- **Runtime**: Node.js
- **Framework**: Express
- **Database**: MongoDB with Mongoose
- **Security**: JWT Authentication

---

## Getting Started

### 1. Prerequisites
- Node.js (v18 or later recommended)
- npm or yarn
- Expo Go app on your mobile device (for frontend testing)
- A running MongoDB instance (local or Atlas)

### 2. Setup

To get the whole system running, follow the instructions in each project's directory:

1.  **Backend**: Go to the `/ShadesX-Server` directory, install dependencies, and start the server.
2.  **Frontend**: Go to the `/ShadesX.` directory, install dependencies, and start the Expo dev server.

Refer to the individual `README.md` files in each folder for detailed setup steps and environment variable configuration (`.env.example`).
